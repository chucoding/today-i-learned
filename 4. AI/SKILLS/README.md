# SKILL

## SKILL 활용

### 플랜모드
순수 플랜 품질만 보면 Cursor 단독 플랜모드 사용보다 Superpowers 또는 gstack을 얹은 Claude Code가 더 강함

#### 프론트엔드 기능 개발
Cursor + Superpowers + grill-me
모델은 Sonnet 4.6 또는 GPT-5.5, 생각수준은 Plan high / 구현 medium~high.

#### 중요한 기능 설계
Claude Code opusplan + gstack + grill-me
모델은 Opus 4.7 → Sonnet 4.6, 생각수준은 xhigh, 최종 리뷰만 max/ultrathink.

#### 단순 구현
Codex GPT-5.5 + Superpowers + grill-me
생각수준은 high, 긴 작업만 xhigh.

#### 고위험 스펙 기능
Ouroboros + Claude Opus/Codex GPT-5.5
생각수준은 spec xhigh~max / 구현 high.
