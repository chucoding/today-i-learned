## 채팅

모든 채팅

### Thinking (GPT-5.4)
복잡한 작업을 위한 깊은 추론

#### 특징
- 다단계 추론, 긴 문맥, 도구를 많이 쓰는 워크플로를 비교적 효율적이고 낮은 지연으로 처리하도록 설계 
- 대부분의 어려운 일에 쓰는 기본 고급형
- 문서 분석, 코드 수정, 조사, 슬라이드/스프레드시트 같은 대부분의 복잡한 작업은 Pro 대신 Thinking이 더 실용적

#### 컨텍스트
-  256,000 (입력 128,000 + 최대 출력 128,000)

#### 생각 수준
- 라이트 모드
  - 가장 빠른 응답 속도
  - PRO 모드에서만 제공
- 표준
  - 추론 능력을 유지하면서도 더 빠른 답변을 제공하여 속도와 지능의 균형을 맞춤
- 확장
  - 중요도가 높거나 질문이 어려울 때, 더 깊고 포괄적인 답변을 할 수 있도록 더 많은 시간을 제공
- 헤비 모드
  - 더 심층적인 추론 (약 5분 정도)
  - PRO 모드에서만 제공

### Pro (GPT-5.4)
리서치급 인텔리전스

#### 특징
- Thinking에 비해 더 많은 연산을 써서 더 오래/깊게 생각하며 더 일관된 고품질 답변을 내도록 설계
- 속도보다 품질과 깊이를 우선하는 최상위형
- 임원 보고용 메모처럼 결과를 여러 번 수정하기 어렵거나 아주 까다로운 연구·검증 작업처럼 한 번의 품질이 중요한 경우에 사용(OpenAI도 Pro를 “결정에 바로 쓸 수 있고 수정 비용이 큰 결과물”에 적합하다고 안내)
- 앱, 메모리, 캔버스 및 이미지 생성 기능을 사용 불가능
- 프로 등급: 400k (입력 272k + 최대 출력 128k)

#### Pro 생각 강도
- 표준
- 확장 : 25분 정도 소요

> 💡 생각강도  
> - 샘플 수/빔 수가 곧 테스트 시간 계산량을 선형(혹은 그 이상)으로 증가시킴
> - “한 번의 답”이 아니라 “여러 후보를 보고 고른 답”이 되므로 사용자 체감으로는 “더 고민하고 답한다”에 가깝다. 

## 심층 리서치
보고서형 조사 워크플로우

### 특징
시작 전에 어떤 소스(웹사이트, 업로드 파일, 연결 앱)를 쓸지 정하고, ChatGPT가 만든 연구 계획을 미리 검토·수정한 뒤 실행

> 💡 심층 리서치 vs PRO 채팅  
> - 심층 리서치 : 웹+내 파일+연결 앱까지 묶어서 검토 가능한 결과물이 필요한 경우에 사용
> - GTP 5.4 PRO 채팅 : 최신 자료 리서치
