# SENS(Simple & Easy Notification Service)
SMS, PUSH, 알림톡 등 메시지 알림 기능을 구현하는 서비스

# 요금
PUSH : 2000건 이하 무료<br/>
SMS : 50건 이하 무료<br/>
LMS : 10건 이하 무료<br/>

# SENS 등록 방법
1. 사용 권한 설정
2. 프로젝트 생성
3. 사전(서비스 Type) 설정(Push, SMS 등)
4. 메시지 발송
5. 메시지 발송 이력 조회

# 인증서 등록
SENS중 PUSH 이벤트를 사용하려면 OS 인증서가 필요
- Android : FCM(Firebase Cloud Messaging)
- IOS : APNS
