# 프로젝트 구조
## 예시
### 지그재그 - 카카오스타일
```
ZDS (Zigzag Design System): 각 모듈을 포함하는 메인 패키지
 |- Core: 로깅 및 전반적인 코어 기능을 제공(Logger, Etc)
 |- Components: 디자인 시스템 컴포넌트 제공(Button, Product Card)
 |- Icons: 디자인 시스템 아이콘 에셋 제공(ArrowUpBold, ArrowUpRegular)
 |- Themes: 서비스의 테마 스타일 제공(Light Theme, Dark Theme)
 |- Tokens: 디자인 시스템에 정의된 디자인 토큰 제공(Base Tokens, Semantic Tokens)
```


### tokens
디자인 토큰 정의

```
tokens
  |- color.css.ts        /* 컬러 토큰 */
  |- index.css.ts
  |- palette.css.ts      /* 색상 팔레트 */
  |- spacing.css.ts
  |- typography.css.ts
```

### 진입점 설정

다른 서비스에서 빌드된 모듈을 다음과 같이 사용할 수 있도록 설정 

```tsx
import { Button } from '@rsquare/ds';
```

진입점 설정

```json
{
  "main": "dist/index.mjs",
  "module": "dist/index.mjs",
  "types": "dist/index.d.ts",
  "exports": {
    ".": {
      "import": "./dist/index.mjs",
      "types": "./dist/index.d.ts"
    },
    "./dist/index.css": "./dist/index.css"
  },
}
```

컴포넌트별 진입점이 필요한 경우

```json
{
  "exports": {
    ".": "./dist/index.mjs",
    "./button": "./dist/components/Button/index.mjs",
    "./input": "./dist/components/Input/index.mjs"
  }
}
```
