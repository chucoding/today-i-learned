# Fetch
API를 호출하고 응답 데이터를 받아옴
```js
const form = new FormData(...);
fetch("https://jsonplaceholder.typicode.com/posts", {method:'POST', body:form})
.then(res => res.json())
.then(json => console.log(json));
```
